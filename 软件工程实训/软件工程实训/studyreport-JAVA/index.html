  • 表明方法或属性是与类绑定的, 而不是与对象绑定的. 一个类的所有对象共享一个静态属性, 且静态属性通过类名调用

读写文件

有常见的几种流:

  • 字节流: 以8位字节为单位进行读写, 以InputStream与OutputStream为基础类
  • 字符流: 以16位字符为单位进行读写, 以Reader与Writer为基础类
  • 文件流: 对文件进行读写和传输, 使用FileOutputStream与FileInputStream. 它们是InputStream与OutputStream的子类

序列化与反序列化

  • 序列化: 将对象转换成串行格式以便保存在硬盘上. 基础类是ObjectOutputStream
  • 反序列化: 将序列化的串行格式从硬盘中读取后, 恢复成对象. 基础类是ObjectInputStream
