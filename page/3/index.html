发表于 2018-12-29 | 分类于 环境配置

Node.js

  1. 去官网下载二进制包(注意是.xz结尾的二进制包, 不是以.gz结尾的源码包)
  2. 通过FileZilla上传到云服务器的/usr/local/tools文件夹下
  3. tar -xvf node-v10.15.0-linux.xz 以解压
  4. mv node-10.15.0/ ../node/ 将解压后的文件夹移入到node文件夹
  5. 通过vi修改/etc/profile文件, 在结尾添加
  1. 运行source /etc/profile
  2. 在根目录下运行node -v以查看结果

MongoDb

参考

​ https://www.cnblogs.com/xiaohuochai/p/7636245.html

安装

  1. 官网下载Mongodb的二进制包(.gz后缀)
  2. 通过FileZilla上传到云服务器/usr/local/tools下
  3. 在tools文件夹下解压
  4. 创建/usr/local/mongodb, 然后运行mv命令将解压后的文件夹放入其中
  5. mongodb文件夹下创建conf和logs文件夹
  6. 在logs文件夹下, 通过命令touch mongodb.log生成日志文件
  7. 在conf文件夹下, 通过命令vi mongodb.conf配置
  1. 在根目录下创建/data/db文件夹
  2. 执行export PATH=/usr/local/mongodb/bin:$PATH 添加路径
  3. 创建软链接
  1. 运行mongod -f /usr/local/mongodb/conf/mongodb.conf开启mongodb数据库
  2. 创建一个新的终端, 输入mongo以验证结果
